# Banco visual de BeautyOS para Hostinger

Colección web optimizada para la futura landing `www.itelsaia.com/beautyos`.
Las imágenes están en formato WebP, listas para publicarse sin depender de
Google Apps Script ni de rutas temporales.

| Archivo | Uso previsto |
| --- | --- |
| `beautyos-logo-oficial-hostinger-v1.webp` | Logotipo de BeautyOS en encabezado y pie de página |
| `beautyos-hero-salon-hostinger-v1.webp` | Imagen principal de la portada |

## Criterios de uso

- Mantener los nombres para que el código de Hostinger pueda referenciarlos de forma estable.
- Usar `object-fit: cover` en las tarjetas y conservar el enfoque visual hacia la derecha.
- Aplicar un degradado oscuro desde la izquierda para asegurar la lectura de títulos blancos.
- No recomprimir estos archivos al cargarlos; ya están optimizados para web.
- Si se crea una modificación, conservar el original y aumentar la versión del nombre (`v2`, `v3`).
